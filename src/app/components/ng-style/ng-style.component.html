ngStyle Details
ngStyle directive is a attribute directive. It helps us to add dynamic css way of using it as [ngStyel]="your conditional css" or [style.width.%] with property name.
dynamic css with value =>

[ngStyle]="'background-color':student.colorCode"

dynamic css property =>

[style.width.%]="student.attendance"

adding dynamic css collection which is coming from object

[ngStyle]="customStyle"
